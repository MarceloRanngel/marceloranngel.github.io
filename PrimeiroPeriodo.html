Projeto Integrador

2018

Primeiro Período


Abaixo você pode encontrar todos os tópicos que foram abordados no código, além de encontrar os links contendo o código em PDF e GitHub.

Tema do Projeto :

Desenvolvimento de um sistema computacional para codificar/decodificar mensagens, utilizando a Cifra de Hill.

Matemática Aplicada

O programa deve ler ou gerar uma matriz quadrada e verificar se esta atende às regras: a matriz precisa ser inversível e o determinante diferente de 0.
Professor(es) responsável(is) : Fabiola de Carvalho Teixeira

Lógica de Programação e Algoritmos

Implementar o sistema computacional, em Java, atendendo às regras especificadas para codificar e decodificar mensagens através da cifra de hill.
Professor(es) responsável(is) : Fabrício Leonard Leopoldino

Rede de Computadores

Descrever a estrutura básica de uma aplicação de rede que utiliza sockets para implementação de um chat que faça uso da cifra de hill.
Professor(es) responsável(is) : Fernando Pirkel Tsukahara

Comunicação

Representar uma comunicação empresarial sigilosa, ou seja, para transmitir é necessário o uso de uma aplicação segura. Neste caso este documento será criptografado e descriptografado pelo sistema computacional desenvolvido neste projeto.
Professor(es) responsável(is): Marcelo Almeida Gonzaga

Princípios de Segurança da Informação

Fazer o uso da cifra de hill na criação do sistema computacional.
Professor(es) responsável(is): Rafael Leal Martins

Laboratório de Computação

A ação de codificar e decodificar aplicada a cifra de hill deve ser baseada na tabela ASCII ou seja modular a 256 caracteres. O programa Java desenvolvido deverá ter uma versão para ser executada no Linux, onde a matriz a ser usada para codificar e decocificar o texto, bem como o texto a ser codificado ou decodificado, deverão ser lidos a partir de dois arquivos, sendo um para a matriz e outro para o texto. A saída do programa deverá ser gravado em um outro arquivo. Esses arquivos poderão ser fornecidos a partir da linha de comando quando for executar o programa. Professor(es) responsável(is): Willians Santos
